# Iconos pixel-art (estilo Pokémon / Tibia)

Coloca aquí **GIF** (o PNG) pixelados para las cartas. Si falta un archivo, el juego mostrará el emoji por defecto.

## Formato recomendado
- **Estilo:** pixel art mitológico, 16x16 o 32x32 px, estilo Tibia/Pokémon.
- **Formato:** `.gif` (animado o estático). También puedes usar `.png` cambiando la extensión en `ui.js` (`getCardIconUrl`).
- **Fondo:** transparente.
